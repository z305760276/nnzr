CRM工单流转全景图

基于《客户服务部管理制度》V2.0 第三章 工单管理制度

5大流转阶段 5个责任岗位 3级隐患处理 3条专家经验
01
STEP 01 工单发起 （即时生成≤3分钟）

用户通过95007热线、微信公众号、营业厅、全国联络中心等渠道提交服务请求，系统自动生成CRM工单并记录诉求详情。

责任岗位
全国联络中心接线员 网厅运营专员 营业厅前台
系统操作要点
  1. CRM系统自动建单
  2. 诉求分类标记（维修/安检/通气/投诉）
  3. 关联用户档案调取历史记录
  4. 生成工单编号并短信通知用户
来源：《客户服务部管理制度》V2.0 第三章 第1条
02
STEP 02 工单派发 （普通24h内 / 抢险即时）

服务运营监督组审核工单信息完整性，根据服务类型和地理位置匹配最近服务站或网格员。派发前需确认网格员当日负荷，避免超负荷派单。

责任岗位
服务运营监督班长 服务站站长 网格长
系统操作要点
  1. CRM派单至服务站
  2. 服务站匹配最近网格员（GPS距离 + 技能标签）
  3. 网格员移动端接收工单提醒
  4. 系统自动发送预约短信给用户
来源：《客户服务部管理制度》V2.0 第三章 第2条
03
STEP 03 上门服务 （安检3日 / 维修24h / 抢险30分钟）

网格员按约定时间上门，提前30分钟致电用户确认。到达后出示工作证，穿戴鞋套进入。执行标准化作业流程，全程使用手持终端记录，关键节点拍照上传CRM。

责任岗位
网格员（开通维修岗） 安检员 安全技术员（复杂作业）
系统操作要点
  1. 手持终端签到（GPS定位）
  2. 作业执行（按标准SOP）
  3. 关键节点拍照上传（至少8张）
  4. 用户电子签名确认
  5. 离场关阀检查
来源：《客户服务部管理制度》V2.0 第三章 第3条
04
STEP 04 隐患处理 （一级1~7日 / 二级30日 / 三级30日）

发现隐患按三级分级标准处理。一级隐患（重大）立即关阀或停气报调度中心；二级隐患（一般）现场签发整改通知单；三级隐患（轻微）现场口头告知并发放宣传资料。

🔴 一级隐患 · 重大
漏气、直通表等严重隐患。
立即关阀或停气，30分钟内上报调度中心。CRM标记红色紧急+自动上报。
⏱ 1-7日内整改完毕
🟠 二级隐患 · 一般
软管老化等常规隐患。
现场签发整改通知单，用户签字确认。生成整改跟踪记录。
⏱ 30日内整改完毕
🔵 三级隐患 · 轻微
轻微问题，现场口头告知。
拍照留痕+发放宣传资料，下次安检时重点关注复查。
⏱ 建议30日内改善
责任岗位
网格员 安全技术员（技术判定） 安检网格长（审核）
来源：《客户服务部管理制度》V2.0 第三章 第4条
05
STEP 05 服务回访 （短信即时 / 人工48h）

服务完成后系统自动触发短信回访，询问用户满意度（1-5分）。不满意工单（≤3分）自动转入二次跟踪队列，由服务运营监督组人工回访了解具体问题。所有回访记录存档备查。

责任岗位
服务运营监督组 系统自动触发短信
系统操作要点
  1. 服务完成标记后自动触发短信回访
  2. 满意度统计（1-5分）
  3. 差评（≤3分）自动标记红色预警
  4. 人工二次回访
  5. 归档生成服务质量周报
来源：《客户服务部管理制度》V2.0 第三章 第5条

专家经验 · 老师傅传帮带

3条实操经验 —— 场景 + 土办法 + 制度依据 + 风险提示

01
老旧小区户内管道锈蚀判定
土办法 / 实操技巧
用硬币轻刮锈蚀表面，若锈层脱落露出金属光泽为"中度锈蚀"；若刮后呈坑洼状且管壁变薄为"严重锈蚀"，需立即建议用户更换。切勿仅凭外观判断。
制度依据
《隐患管理制度》表5 锈蚀等级3-4级标准
风险提示
误判为轻微锈蚀可能导致后期漏气事故，造成人身伤亡。锈蚀判定错误属于严重工作失误。
02
用户不在家时的安检处理
土办法 / 实操技巧
粘贴《到访不遇告知单》后拍照上传CRM（含门牌号、告知单张贴位置），同时在门口张贴安全用气宣传单。下次安检时此类用户优先安排，提前电话确认。
制度依据
《安检管理制度》第三章第2条：到访不遇处理规范
风险提示
未留痕可能导致安检入户率统计失真，外包结算产生争议。用户后续投诉未通知到门，公司需承担举证责任。
03
CRM系统卡顿时的工单处理
土办法 / 实操技巧
手机端APP可离线缓存最近30条工单数据，网络恢复后自动同步。若抢修工单无法及时录入，可先电话直报调度中心（3102277），调度中心代为补录，网格员返岗后24小时内补全系统记录。
制度依据
《管理制度》第六章 信息系统管理第12条：系统故障应急处理
风险提示
离线操作须当日补录，超期未补录按工单缺失考核（50元/次）。抢险工单未录入导致事故追溯困难，责任由网格员承担。
